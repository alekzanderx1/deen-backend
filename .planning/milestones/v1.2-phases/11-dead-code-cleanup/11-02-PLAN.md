## Context

`tests/test_embedding_service.py` was written for the old `EmbeddingService` that used `OpenAI(api_key=...)` and `client.embeddings.create()` returning 1536-dim vectors.

After Phase 10, `EmbeddingService.__init__` calls `getDenseEmbedder()` (HuggingFace, 768-dim). The two fixtures — `mock_openai_client` and `embedding_service` — patch `services.embedding_service.OpenAI` which no longer exists in that module, causing `AttributeError` at test collection time.

The fix: replace `mock_openai_client` with `mock_embedder` that patches `services.embedding_service.getDenseEmbedder`, and update all 768-dim assertions.
